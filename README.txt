=============================
 CELSOS - Sistema de Emergencia Personal
=============================

1. Abre esta carpeta en cualquier PC que tenga Python.
2. Entra a 'src/backend/' y ejecuta este comando:

   python app.py

3. Abre tu navegador y entra a:

   http://localhost:5000

4. Inicia sesión con:

   Usuario: admin
   Contraseña: celsos123

5. Elige un tema, envía alertas y prueba todas las funciones.

¡Tu seguridad, en un solo clic!
